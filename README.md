# SOCNITI

SOCNITI is a social impact platform for discovering nearby NGO events, registering for them, chatting with other attendees, donating to community needs, and helping organizers manage participation.

## Current baseline

- React + Tailwind frontend shell
- API gateway
- Auth microservice
- Event microservice
- Docker and environment templates

## Quick start

1. Copy `.env.example` to `.env`
2. Add your MongoDB, JWT, SMTP, and map credentials
3. Run `npm install`
4. Run `npm run dev`
